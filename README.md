# Introduction

此项目为学习大模型相关知识创建的仓库，仓库中的代码都来自开源社区，只添加代码注释，新增相关知识学习笔记。目标是熟悉大模型预训练阶段（Pre-training），后训练阶段（Post-training），推理部署，量化等技术细节！
